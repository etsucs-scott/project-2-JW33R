How To Build Game:
git clone https://github.com/etsucs-scott/project-2-JW33R.git 
dotnet restore #Used to install any dependencies if you don't have them 
dotnet build 
dotnet run
---
The Player Count is chosen by the Person playing the game. A simple input prompt is shown and the person has to enter a number
